# 👾 Glyph Invaders

A Space Invaders-style game running entirely on the **Nothing Phone Glyph Matrix**

---

## 📱 Compatible Devices

- Nothing Phone 3
  
## 🎮 How to Play

The game is controlled by **tilting your phone**:

| Action | Control |
|---|---|
| Move left | Tilt phone left |
| Move right | Tilt phone right |

- Survive waves of enemies 👾
- Defeat the Bosses 💥
- Have fun 😊 

---

## ⬇️ Installation

1. Download the latest `app-release.apk` from the [Releases](../../releases) section
2. On your Nothing Phone, go to **Settings → Apps → Install unknown apps**
3. Allow installation from your browser or file manager
4. Open the APK and install
5. Open **Glyph** app → Toys → **Glyph Invaders**

> ⚠️ Google Play Protect may show a warning – this is normal for sideloaded apps. Tap **"Install anyway"** to continue.
